# 어디 갈래 — 레이아웃 구조도 v3

## 1. UI 설계 원리: "나경수(羅經樹)의 보물지도" + 나린(羅鱗)

여행 + 아기 용 = **보물지도**가 화면의 물리적 중심.
나린(귀여운 아기 용)이 나침반 위에서 날아올라 방위를 찾고, 여행지를 발견하는 경험.

**핵심 메타포:**
- 나침반 = 나린의 가슴에 새겨진 나침반 문양의 확대판
- 결과 = 보물지도가 펼쳐지며 방위별 여행지가 드러남
- 도장판 = 5방위 용 발자국 (방위를 방문하면 발자국이 찍힘)
- 여행 기록 = 나린의 비행 궤적 (지도 위의 점선 경로)

---

## 2. 3대 원칙 적용표

| 원칙 | 적용 |
|------|------|
| 주제 매칭 | 여행+용 → **두루마리 보물지도**가 화면 중심. 나린가 지도를 물고 날아다니며 여행지 탐색 |
| 비정형 배치 | 보물지도(가장 큼) → 나침반 → 용 발자국 도장판 → 여행지 스크롤 → 팁 카드 순으로 시각 무게 감소 |
| 정보 덩어리화 | 여행지(장소+음식+활동+계절) → **1문장 모험 시나리오**로 통합 서술 |

### 점수/등급 대체

| 금지 | 치환 |
|------|------|
| S/A/B/C 등급 | **용의 발자국 도장 (dragon footprint stamp)** — 방위별 다른 색 발자국 |
| 87점, 퍼센트 | **나침반 불꽃 glow** — 활성 방위에서 나린의 불꽃이 타오름 |

## 3. 마스코트: 나린 (羅鱗)

나경수(羅經樹)의 마지막 알에서 태어난 아기 용. chibi 비율(머리:몸 = 1:1).
작은 뿔, 짧은 날개, 뭉뚝한 꼬리, 큰 눈(나침반 금색 하이라이트).
**전체 백색 톤온톤** — 아이보리, 펄화이트, 크림, 실버그레이 등 명도/채도 차이로 디테일.
수묵화풍 터치의 부드러운 윤곽선.

### 마스코트 포즈 활용표

| 포즈 | 이름 | 크기 | 용도 | 표현 |
|------|------|------|------|------|
| main (앉기) | sitting | 120px | 홈 화면, 나침반 위 | 나침반 위에 배 깔고 앉아 있음, 꼬리 흔들기 |
| flying (날기) | flying | 180px 폭 | 나침반 회전 중 | 작은 날개로 퍼덕이며 나침반 주위 비행, 불씨 파티클 |
| thinking (생각) | thinking | 64px | 로딩/분석 | 고개 갸웃, 머리 위 물음표 연기, 발로 턱 짚기 |
| guiding (안내) | guiding | 80px | 결과 여행지 안내 | 앞발로 방향 가리키며 날개 펼침, 입에서 불꽃 화살표 |
| happy (기쁨) | happy | 72px | 도장 수집/프리미엄 해금 | 양 팔 들고 점프, 머리 뿔에서 별 파티클 |

- 말투: ~래요, ~거든래요, 크르릉~ (나침반이 알려주는 걸 전달 + 아기 용 으르렁)
- 가슴에 나침반 문양 (비활성: 은색, 활성: 금색 발광)
- 방위를 방문하면 용 발자국 도장이 찍힘 (5방위 수집)

---

## 4. 핵심 차별점: "이름 입력 없는 나침반 탭 인터랙션"

- 기존 모든 앱: 이름 입력 → reveal → result (텍스트 입력 기반)
- **어디 갈래: 나침반 탭 → 나린가 불을 뿜음 → 나침반 회전 → 방위 착지 → 보물지도 펼침**
- 입력 필드 완전 제거 → deviceId + 날짜 기반 결과 결정
- **세로 스크롤이 아닌 "지도 펼침" 트랜지션** — 중앙에서 사방으로 확장

## 6. 고유 컴포넌트: CompassRose SVG

```
         北 (검은색)
          ^
         / \
   西   /   \   東
(흰색) o---o---o (청록)
        \   /
         \ /
          v
         南 (주황색)

  중앙: 중(中) 금색 원
  나린: 앉기 시 중앙 위에 앉음
  불꽃: 회전 시 바늘 끝에서 불꽃 잔상
```

SVG (viewBox 300x300):
- 외곽 원 (r=130) — 점선 테두리 (보물지도 느낌)
- 내곽 원 (r=115)
- 5방위 포인터 (각 방위 색상, 한글 라벨)
- 중앙 원 (금색, 中 문자)
- 나침반 바늘 (활성 방위를 가리킴, 스핀 후에만 표시)
- 바늘 끝에 용 불꽃 이펙트

**방위 색상 (v3 — 자연 테마):**

| 방위 | 색상 | 테마 | Hex |
|------|------|------|-----|
| 東 (동) | 청록 | 숲/산 | #2DD4BF |
| 西 (서) | 흰색/은색 | 바다/섬 | #E2E8F0 |
| 南 (남) | 주황 | 태양/남쪽 | #F97316 |
| 北 (북) | 검정/진회 | 밤하늘/북쪽 | #1E293B |
| 中 (중) | 금색 | 중심/보물 | #F59E0B |

**활성 상태** (방위 찾은 후):
- 해당 방위: scale 1.4 + 불꽃 glow filter + pulse
- 다른 방위: opacity 0.1
- 나린 뿔 → 활성 방위색으로 빛남
- 바늘 끝에서 용 불꽃 파티클

**비활성 상태** (홈 화면):
- 모든 방위: opacity 0.25, 균등 크기
- 바늘 없음, 나린가 중앙에 앉아 있음
- 점선 테두리 약하게 pulse

## 7. 비정형 배치 원칙표

| 요소 | 시각 무게 | 크기 | 이유 |
|------|----------|------|------|
| 보물지도 (모험 시나리오 포함) | ★★★★★ | 전폭, 패딩 강조 | 최고 결과 — 여행 스토리를 한눈에 |
| 나침반 (sticky) | ★★★★ | 160px (축소) | 방위를 계속 상기, 나린가 위에 앉음 |
| 용 발자국 도장 | ★★★★ | 160px 원형 | 어떤 방위인지 강렬하게 — stampPress 불꽃 |
| 국내 여행지 카드 | ★★★ | 가로 스크롤 비정형 카드 | 핵심: 어디로 가야 하는지 (큰 카드+작은 카드 혼합) |
| 나린 말풍선 | ★★ | 중형 | 가이드 캐릭터 조언 (guiding 포즈) |
| 프리미엄 (잠금) | ★ | 잠금 카드 2개 | 부가 기능 (광고 게이트) |
| 도장판 (상단) | ★ | 소형, 상단 좌측 | 진행률 표시 (간결하게) |
| 여행 기록 | ★ | 소형, 하단 타임라인 | 가로 스크롤 미니 아이콘 |

## 8. 애니메이션 시스템 (Framer Motion)

| 이름 | 용도 | 도구 | 동작 |
|------|------|------|------|
| compassSpin | 나침반 회전 | useAnimate | 3바퀴 빠르게 → 감속 → 목표 각도 정지 (2.2s) |
| dragonFlight | 나린 비행 | keyframes | 나침반 주위 원형 궤도 비행 (1.6s) |
| fireBreath | 불꽃 스트림 | spring | 입에서 나침반 방향으로 불꽃 파티클 (0.8s) |
| flameGlow | 방위 활성화 | spring | blur+scale 0.5→1.4→1.0 + 불꽃 색상 (0.8s) |
| stampPress | 용 발자국 찍기 | spring | scale 2.5→0.85→1.0 + rotate + 불꽃 burst (0.7s) |
| mapUnfurl | 보물지도 펼치기 | AnimatePresence | scale 0→1 + opacity (1.0s), 양피지 질감 |
| breathe | 나린 대기 호흡 | repeat | scale(1→1.03→1) + translateY(0→-4px→0) 반복 (2.5s) |
| tailWag | 꼬리 흔들기 | repeat | rotateZ(-15→15→-15) 반복 (1.8s) |
| fadeIn / slideUp | 기본 전환 | variants | 표준 패턴, stagger 120ms |
