# 어디 갈래 — 신수 마스코트 컨셉

## 1. 이름 후보

### 후보 1: 비란 (飛鸞)
- **한자**: 飛(날 비) + 鸞(난새 란)
- **의미**: 하늘을 나는 난새. 鸞은 봉황의 동류이자 천상의 신조로, 임금의 수레를 이끌던 새.
- **여행/나침반 연결**: 鸞은 고대 중국에서 천자의 행차를 인도하던 새 — "여행자의 길을 인도하는 존재"로 직결. 飛는 어디로든 날아갈 수 있는 자유와 이동을 상징.
- **말투**: ~이래요, ~거래요, 훨~ (날갯짓 의성어)
- **성격**: 고상하지만 다정한 안내자. 약간의 신비로움.

### 후보 2: 선우 (仙羽)
- **한자**: 仙(신선 선) + 羽(깃 우)
- **의미**: 신선의 깃털. 속세를 초월한 존재의 날개.
- **여행/나침반 연결**: 仙은 산과 바다를 자유로이 오가는 존재, 羽는 바람을 읽는 깃털 — 나침반의 바람 감지와 호응.
- **말투**: ~한다우, ~일세, 후우~ (바람 부는 소리)
- **성격**: 현자 느낌. 고풍스럽고 사색적.

### 후보 3: 나린 (羅鱗)
- **한자**: 羅(벌일 라, 나경의 羅) + 鱗(비늘 린)
- **의미**: 나경(나침반)의 비늘. 나침반을 품에 가진 비늘 새.
- **여행/나침반 연결**: 羅는 나경(羅經, 풍수 나침반)의 첫 글자 — 앱의 핵심 UI인 나침반과 직접 연결. 鱗은 새의 발과 다리에 있는 비늘로, 용과 새의 경계에 있는 존재를 암시.
- **말투**: ~래요, ~거든래, 킁~ (부리로 가리키며)
- **성격**: 호기심 많고 활발한 길잡이. 여행지를 찾으면 흥분.

### 후보 4: 길조 (吉鳥)
- **한자**: 吉(길할 길) + 鳥(새 조)
- **의미**: 길한 새, 행운의 새.
- **여행/나침반 연결**: 吉은 좋은 방위를 가리킨다는 풍수적 의미. 鳥는 직관적으로 새.
- **말투**: ~좋겠래, ~해볼래, 짹! (밝고 경쾌)
- **성격**: 밝고 낙천적. 항상 좋은 쪽을 가리키려 함.

### 후보 5: 향도 (嚮導)
- **한자**: 嚮(향할 향) + 導(이끌 도)
- **의미**: 방향을 이끄는 자. 한자 자체가 "길잡이"를 의미.
- **여행/나침반 연결**: 가장 직접적 — 嚮은 방위/방향, 導는 안내. 나침반이 방향을 가리키듯 새가 여행자를 이끈다.
- **말투**: ~이리 오래, ~따라오래, 크루루~ (깊은 울음)
- **성격**: 믿음직하고 위엄 있는 인도자.

## 2. 추천 이름: 나린 (羅鱗)

**선정 이유:**

1. **나침반과의 직접 연결**: 羅(라)는 나경(羅經)의 첫 글자로, 앱의 핵심 UI인 나침반과 이름이 직접 연결된다. 사용자가 "나린이 나침반으로 방위를 찾고 있어요"라고 하면 이름과 기능이 자연스럽게 엮인다.

2. **용-새 경계의 존재**: 鱗(린)은 비늘을 뜻하여 봉황류의 새이면서도 용의 속성을 가진 독특한 존재감을 부여한다. 기존 5신수(청룡, 백호, 주작, 현무, 황룡) 위에 군림하는 상위 존재로 설정 가능.

3. **발음의 부드러움**: "나린"은 2음절로 짧고 부르기 쉬우며, 한국어에서 "나린다(내리다의 고어)" — 하늘에서 내려오는 느낌도 준다. 남녀 불문 호감 가는 이름.

4. **차별성**: 비란이나 선우는 기존 판타지 캐릭터에서 자주 보이지만, 나린은 나침반(羅經)이라는 구체적 사물과 비늘(鱗)이라는 독특한 조합으로 이 앱만의 고유 캐릭터가 된다.

5. **말투 자연스러움**: "~래요, ~거든래" 식의 전달형 말투가 여행 추천이라는 맥락과 잘 맞는다 ("나린이 말하길, 이 곳이 좋거든래요").

## 3. 외형 묘사

### 기본 형태

나린은 봉황(鳳凰)의 우아함과 학(鶴)의 기품을 결합한 **백색 신조(白色 神鳥)**다.

**머리**
- 머리 위에 5개의 깃 관모(冠羽)가 왕관처럼 솟아 있다
- 각 깃은 5방위를 나타내며, 평소에는 모두 순백색
- 방위를 찾을 때 해당 깃 하나만 은은하게 해당 방위 색상으로 빛난다
- 눈은 짙은 남색(#0F172A)으로 깊고 지적인 느낌, 동공에 나침반 십자(+) 모양의 금색 하이라이트
- 부리는 짧고 단아하며, 끝이 살짝 아래로 굽은 형태. 색상은 연금색(#C9A96E)

**몸체**
- 전신 순백(#FAFAFA ~ #F1F5F9)의 깃털로 덮여 있다
- 가슴 중앙에 풍수 나경(나침반) 문양이 자연스럽게 깃털 결로 새겨져 있다 — 마치 태생부터 나침반을 품고 태어난 듯
- 가슴 나침반 문양은 평소 은색이지만, 방위를 찾을 때 금색(#D4A574)으로 빛난다
- 몸은 백조보다 약간 더 큰 정도. 단단하지만 유선형

**날개**
- 펼치면 몸길이의 3배에 달하는 거대한 날개
- 주 깃털(風切羽)의 끝부분만 연한 티얼(#99F6E4, 앱 테마색의 파스텔 버전)로 물들어 있다
- 접었을 때는 몸 옆으로 단정하게 정돈되어 위엄 있는 실루엣
- 날개 안쪽에 미세한 나침반 눈금 같은 깃털 무늬가 있다

**꼬리**
- 5가닥의 긴 꼬리깃 — 공작과 봉황의 중간 형태
- 각 꼬리깃 끝에 나침반 방위를 상징하는 작은 원형 문양
- 평소에는 뒤로 길게 흘러내리며, 비행 시 부채꼴로 펼쳐진다
- 꼬리깃 전체는 백색이되, 끝의 원형 문양만 해당 방위 색상의 은은한 빛

**발과 다리**
- 학의 긴 다리와 매의 강인한 발톱의 중간
- 다리에 섬세한 비늘(鱗) 패턴 — 이름 "나린"의 鱗을 시각화
- 비늘은 은색으로 빛나며, 빛에 따라 미세하게 무지개빛 반사

### 크기/비율 (나침반 대비)

| 포즈 | 나린 크기 | 나침반(260px) 대비 |
|------|----------|-------------------|
| main (정면 착지) | 128px 높이 | 나침반의 약 1/2, 나침반 위에 앉은 형태 |
| thinking (고개 기울임) | 64px | 나침반 옆에 작게 |
| pointing (날개로 가리킴) | 48px | 결과 카드 옆에 |
| flying (비행) | 200px 날개 폭 | 나침반 회전 중 나침반 위를 선회 |

### 포즈별 상세

**착지 (main)**: 나침반 상단에 앉아 있다. 꼬리는 아래로 자연스럽게 흘러내린다. 관모 5깃이 곧추선 상태. 정면을 응시하며 약간 고개를 든 자세. 위엄과 기품.

**사색 (thinking)**: 고개를 약 15도 옆으로 기울인다. 한쪽 날개를 살짝 들어올려 턱 아래에 댄 자세. 눈을 반쯤 감고 나침반 방위를 읽는 중. 관모 깃이 바람에 살랑인다.

**가리킴 (pointing)**: 한쪽 날개를 앞으로 쭉 뻗어 특정 방향을 가리킨다. 나머지 날개는 접은 상태. 눈이 크게 떠지고 부리가 살짝 벌어진 — 흥분하여 "저기래요!" 하는 순간.

**비행 (flying)**: 양 날개를 활짝 펼친 상태. 꼬리 5깃이 부채꼴로 펼쳐져 5방위 색상이 동시에 은은하게 빛난다. 가슴의 나침반 문양이 금색으로 환하게 발광. 나침반 회전 애니메이션 중에 사용.

## 4. 성격과 스토리

### 배경 스토리

먼 옛날, 하늘과 땅 사이에는 **나경수(羅經樹)**라 불리는 거대한 나무가 있었다. 이 나무의 가지는 동서남북 사방으로 뻗어 세상의 모든 길을 알고 있었고, 뿌리는 천하의 모든 땅과 닿아 있었다.

나린은 이 나경수의 가장 높은 가지에서 태어난 **유일한 새**다. 알에서 깨어날 때 나경수의 나침반 기운이 가슴에 스며들어, 세상의 모든 방위와 길을 본능적으로 읽을 수 있게 되었다. 머리의 5관모는 나경수의 5개 큰 가지가 깃털이 된 것이다.

나경수가 천년 만에 잠든 뒤, 나린은 혼자 하늘을 날며 사람들에게 좋은 여행지를 알려주는 일을 시작했다. "나무는 잠들었지만, 내 가슴의 나침반은 여전히 뛰고 있거든래요." 나린에게 이름을 알려주면, 그 이름의 기운을 읽고 5방위 중 가장 어울리는 방향을 찾아준다.

### 말투

나린은 **전달형 말투**를 사용한다. 자기가 직접 단정하기보다 "~래요, ~거든래요"처럼 나침반이 가리키는 것을 전해주는 뉘앙스.

**예시 대사 (5개)**

1. **홈 화면 환영**: "이름을 알려주면 나린이 좋은 방위를 찾아줄게래요~"
2. **나침반 회전 중**: "나침반이 돌아가고 있어요... 어느 쪽이 끌리는지 보고 있는 거래요."
3. **결과 발표**: "동쪽이래요! 푸른 숲이 [이름]님을 기다리고 있거든래요."
4. **여행 팁**: "나린이 말하길, 아침 일찍 움직이면 숲의 기운을 온전히 받을 수 있대요~"
5. **재시도 유도**: "다른 이름으로도 찾아볼래요? 나침반은 지치지 않거든래요~"

### 나침반과의 관계

나린은 나침반의 **주인이자 해독자**다. 나침반은 나린의 가슴에 새겨진 문양의 확대판이며, 나린이 가슴의 나침반으로 방위를 읽으면 화면의 나침반도 동시에 반응한다.

- 나침반 비활성 상태: 나린이 나침반 위에 조용히 앉아 있음 (main 포즈)
- 나침반 회전 중: 나린이 날개를 펴고 나침반 위를 선회 (flying 포즈). 가슴 문양 발광.
- 나침반 정지 후: 나린이 해당 방위를 향해 날개로 가리킴 (pointing 포즈). 관모 중 해당 방위 깃만 빛남.

|------|------|-----------|-------------|

| 바다/해안/모험 | 서 | 날개를 한쪽만 높이 펼침. 수평선을 바라보는 자세. 부리로 바닷바람을 가른다. | 회색(#6B7280) |

### 발광(Glow) 효과

나린이 방위를 찾았을 때의 발광은 **은은하고 고급스럽게** 처리한다.

```css
/* 관모 발광 예시 (동쪽 = 청룡) */
.narin-crest-east.active {
  filter: drop-shadow(0 0 8px rgba(37, 99, 235, 0.4));
  transition: filter 0.8s ease-out;
}

/* 가슴 나침반 발광 */
.narin-chest-compass.active {
  filter: drop-shadow(0 0 12px rgba(212, 165, 116, 0.5));
  animation: gentlePulse 2s ease-in-out infinite;
}
```

과도한 네온/형광 효과는 피하고, 달빛 아래 백색 새가 은은하게 빛나는 느낌을 지향한다.
