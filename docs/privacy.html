개인정보처리방침

시행일: 2025년 1월 1일

1. 수집하는 개인정보

운명의 나침반(이하 "앱")은 별도의 회원가입 없이 이용 가능하며, 다음의 정보만을 처리합니다:

  • 기기 정보: 앱 이용 시 자동으로 수집되는 기기 식별자 (광고 목적)
  • 사용 데이터: 앱 내 입력 정보 (기기에만 저장, 서버 전송 없음)

2. 개인정보의 이용 목적

  • 서비스 제공 및 기능 개선
  • 맞춤형 광고 제공 (Google AdMob)

3. 개인정보의 보관

앱에서 입력한 데이터는 사용자의 기기에만 저장되며, 외부 서버로 전송되지 않습니다. 앱 삭제 시 모든 데이터가 삭제됩니다.

4. 제3자 제공

앱은 광고 서비스를 위해 Google AdMob을 사용합니다. Google의 개인정보처리방침은 여기에서 확인할 수 있습니다.

5. 이용자의 권리

이용자는 언제든지 앱을 삭제하여 저장된 데이터를 삭제할 수 있습니다.
